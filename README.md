## How to use IP-Tracer

* `trace -m` to track your own ip address.

* `trace -t target-ip` to track other's ip address for example `ip-tracer -t 203.176.55.72

* `trace` for more information.

* `apt update`

* `cd IP-Tracer`

* `chmod +x install`
